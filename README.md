##  Sobre o Projeto

Single page website com navegação por âncoras, apresentando informações pessoais, projetos, formação acadêmica e contato.

##  Seções

- **Sobre mim** — Apresentação pessoal, experiência e skills
- **Portfólio** — Projetos desenvolvidos com links para o GitHub
- **Formações** — Graduação, cursos, certificações e idiomas
- **Contato** — Formulário de contato e redes sociais

##  Tecnologias

- HTML5
- CSS3
- JavaScript (validação de formulário e interações)
